コントローラでget post put deleteを識別する方法  

``` ruby
# getかどうか
request.get?

# postかどうか
request.post?

# putかどうか
request.put?

# deleteかどうか
request.delete?

# 文字列で判定
request.request_method == "get"

#シンボルで
request.request_method_symbol == :get
```
