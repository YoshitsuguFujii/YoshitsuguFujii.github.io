例えばmodelへの問い合わせ結果をキャッシュに保存していて、after  _saveなどで消したい場合などは  
Rails.cache.delete('キー名')でできる。  

``` ruby
    users = Rails.cache.fetch("users", expires_in: 30.seconds) do
      User.all.to_a
    end

    # 消す
    Rails.cache.delete('users')
```
