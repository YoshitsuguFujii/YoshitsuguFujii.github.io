clearなんてあったんや。  
<http://ref.xaio.jp/ruby/classes/array/clear>  
  
今まで  

``` ruby
array = ["hoge", "fuga"]
array = []
```

  
ってやってたのが、

``` ruby
array = ["hoge", "fuga"]
array.clear # => []
```

  
って書ける。より直感的ですね。  
  
ちなみに文字列もHashもあるみたいです。

``` ruby
hash = {hoge: "fuga"}
hash.clear # {}

str = "fugafufa"
str.clar # => ""
```
