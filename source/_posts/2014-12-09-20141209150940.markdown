今までdeleteでチクチク消したり、rejectで該当キーをひっかけてhash作り直しリしてました。  
  
exceptなんてあったんですね。  

``` ruby
{name: "藤井", age: 34, job: "developer"}.except(:age) # => { :name => "藤井", :job => "developer" }
```
