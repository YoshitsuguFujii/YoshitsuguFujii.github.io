タイトルに意味はないです。  
今日はerbをhamlに一気に変換したいと思います。  

``` ruby
#Gemfile
gem "erb2haml", :group => :development
```

  
``` ruby
bundle install   
rake haml:convert_erbs
 
ERROR:  Could not find html2haml in your PATH. Aborting.
```

  
あれ？  
  
追記

``` ruby
#Gemfile
group :development do
  gem 'html2haml'
  gem 'erb2haml'
end
```
