調べてみたら自分所のDBからFacebookの情報を消すだけのものが多くて、大元のFacebookのアプリ設定から削除するのがあんまり載ってなかったようなきがするので  
  
以下の感じで、連携解除ができます。  

``` ruby
$facebook = new Facebook(array(
'appId' => アプリのapp_id,
'secret' => アプリのapp_secret
));

$facebook->api("/me/permissions", "delete", array('access_token' => ユーザーのアクセストークン));
```

  
珍しくPHPでした。
