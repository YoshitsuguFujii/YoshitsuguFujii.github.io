find  _byのようなダイナミックファインダで例外  
びっくりまーくつければいいみたいです。  

``` ruby
Prefecture.find_by_name!("存在しない件" ) # => ActiveRecord::RecordNotFound
```

  
rails4ではこうかな  

``` ruby
Prefecture.find_by!(name: "存在しない件") 
```
