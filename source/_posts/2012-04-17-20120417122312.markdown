each  _with  _indexは0始まりでインデックスが使える。

``` ruby
["a","b"].each_with_index{|elem,i| puts i} #=> 0 1
```

  
でも連番降ったりとか１始まりが欲しい場合がある。  
そんな場合。

``` ruby
["a","b"].each.with_index(1){|elem,i| puts i} #=> 1 2
```

  
もちろん数値変えればスタートの値も変わる。
